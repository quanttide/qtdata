# 量潮项目经营模拟器

这是一个面向客户视角的游戏化经营模拟器原型，用来演示“客户提出需求 → 平台量化评估 → 报价接单 → 交付验收 → 反馈建议”的完整业务故事。

## 目标

- 用客户能理解的方式讲清楚复杂技术服务。
- 把需求是否值得投入，转化为预算、工期、价值、风险等可量化指标。
- 让团队快速观察客户顾虑，提升报价、评估和需求拆分效率。

## 本地运行

```bash
npm run dev
```

然后打开终端输出的本地地址即可查看原型。当前原型不依赖第三方包，可直接用 Node.js 启动。

## 原型功能

- 输入客户类型、复杂度、业务价值、预算、周期和报价速度。
- 自动计算投资价值得分、报价区间、交付压力、认知负担和验收难度。
- 输出“副官式”接单建议和服务流程建议。
- 适合后续扩展到真实数据采集、需求问卷、CRM 或报价系统。
