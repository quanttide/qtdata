# 资产管理架构

## 系统架构

```
┌─────────────────────────────────────────────┐
│              客户门户 (ixd)                │
└─────────────────────────────────────────────┘
                    │
                    ▼
┌─────────────────────────────────────────────┐
│              资产服务 (Asset)              │
│  - 资产元数据                              │
│  - S3文件引用                              │
│  - 状态流转                                │
└─────────────────────────────────────────────┘
                    │
                    ▼
┌─────────────────────────────────────────────┐
│              S3对象存储                    │
└─────────────────────────────────────────────┘
```

## 资产模型

```
Asset {
  id: string
  type: enum[dataset, processor, document]
  name: string
  s3_key: string        # S3对象存储路径
  version: string
  status: enum[draft, ready, archived]
  created_at: datetime
}
```

直接S3存储，无任何关联关系。
