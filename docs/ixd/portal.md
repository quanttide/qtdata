# 客户门户

## 模块定位

面向客户的Web端门户，让客户实时查看项目进度和交付资产。

## 核心功能

### 1. 项目概览

客户登录后可以看到所有参与项目的列表和状态。

### 2. 进度追踪

展示项目各阶段进展：

- 需求确认
- 数据采集
- 数据清洗
- 质量验收
- 交付完成

### 3. 资产浏览

客户可在线查看交付的资产：

- 数据集：支持预览、下载
- 数据处理器：代码仓库链接
- 数据文档：在线阅读

### 4. 验收流程

- 查看交付清单
- 确认验收意见
- 反馈问题

### 5. 消息通知

- 进度变更通知
- 验收提醒
- 交付完成通知

## 用户角色

- 客户：查看自己项目的进度和资产
- 运营：管理所有客户项目
- 数据工程师：上传和管理交付资产
